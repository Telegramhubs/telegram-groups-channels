# Telegram 频道与群组目录

发现最优质的 Telegram 频道与群组，涵盖加密货币、科技、新闻、教育、游戏等数十个分类。

本项目基于 tgstat.com 等公开数据源，对公开 Telegram 社群进行系统性整理与分类，方便用户发现优质社区。

## 如何使用

本目录收录的频道与群组均来自 Telegram 公开数据。你可以按分类浏览、点击链接直接加入感兴趣的社区。

**寻找特定主题？**

- 访问 [letsTG](https://letstg.com)——支持关键词搜索+分类筛选+语言过滤，可搜索超过 100,000 个 Telegram 社群
- 或直接在 Telegram 内搜索频道/群组名称

## 数据说明

- 数据来源于 Telegram 公开可见的社群信息
- 列表按订阅人数/成员数量排序
- 定期更新，数据截止日期标注于文件名
- 成人内容（赌博、色情等）已被过滤

## 关于本项目

本项目旨在整理 Telegram 公开社群数据，帮助用户发现优质社区。数据经过系统性清洗与分类，去除重复和无效内容。
